# Icarus ModInfo Tool

A desktop app for ICARUS mod authors to create and manage `modinfo.json` files without touching JSON manually.

## Features

- Create new modinfo files from scratch with guided fields
- Edit mod name, description, author, version, and compatibility tags
- Manage multiple mods inside a single modinfo file
- Live JSON preview
- Auto-backup system — never lose work
- Clean dark UI
- Auto-updates — notifies you when a new version is available

## Download

Head to [Releases](../../releases/latest) and download `modinfo_tool.exe`.

No installation needed — just run the exe.

## Requirements

- Windows 10 / 11
- No Python required — fully self-contained executable

## Usage

1. Run `modinfo_tool.exe`
2. Open an existing `modinfo.json` or create a new one
3. Edit your mods and save

## Updates

The tool checks for updates automatically on startup. If a new version is available you'll see a notification bar at the top — click **Update Now** to install without losing any of your files.
